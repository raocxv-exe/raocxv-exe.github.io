# Mi Portafolio Profesional

¡Bienvenido a mi sitio web personal! Esta es la plataforma donde comparto mi trayectoria como desarrollador, mis proyectos más relevantes y mi experiencia profesional.
# https://raocxv-exe.github.io/

## 🌟 ¿Qué es este sitio?

Este sitio web es un **Portafolio Interactivo** diseñado para presentar de manera clara y profesional:

- **📄 Currículum Vitae (CV):** Mi trayectoria profesional y académica (UAZ) disponible en dos idiomas.
- **💻 Proyectos Seleccionados:** Desde apps móviles como **Devii** (Gamificación y Salud Mental) hasta sistemas complejos como **SICZ**.
- **🧠 Mi Filosofía (Why me?):** Por qué mi enfoque combina la ingeniería robusta con la psicología humana y el diseño centrado en el colectivo.


---

## 🛠️ Tecnologías Utilizadas

Este portafolio ha sido construido utilizando tecnologías modernas para garantizar un rendimiento óptimo y una experiencia de usuario fluida:

- **[Astro](https://astro.build/):** El framework principal que permite una carga ultrarrápida (enfocado en el contenido).
- **CSS Vanilla (Custom Styles):** Para un diseño único, artesanal y totalmente responsive.
- **JavaScript Moderno:** Para interactividad y funcionalidades dinámicas como el cambio de tema (Dark/Light mode).
